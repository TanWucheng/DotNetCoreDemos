# DotNetCoreDemos

欢迎下载[我的.Net core示例](https://github.com/TanWucheng/DotNetCoreDemos)仓库

$\small\bold{克隆}$

```shell
git clone https://github.com/TanWucheng/DotNetCoreDemos.git
```

$\small\bold{Restore}$

```shell
dotnet restore
```

$\small\bold{Build\&Run}$

```shell
dotnet build && dotnet run
```
